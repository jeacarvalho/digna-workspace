Pré-visualização do Dossiê

O dossiê CADSOL é um documento oficial para formalização perante o Ministério do Trabalho e Emprego (MTE). Inclui hash SHA256 para auditoria pública e comprovação de integridade.

Clique em "Gerar Dossiê CADSOL" para visualizar o documento.

Importante: Formalização Gradual

O sistema não força a burocracia governamental. A formalização é um processo pedagógico:

  • Grupos com menos de 3 decisões devem focar em construir autogestão
  • Cada decisão registrada no CADSOL é um passo rumo à soberania institucional
  • O hash SHA256 garante auditoria pública imutável das decisões
  • A formalização acontece no tempo do grupo, não no tempo do Estado

📚 Documentação Técnica

Sistema CADSOL

Cadastro de Decisões Soberanas - Registro imutável de deliberações de assembleia com hash SHA256 para auditoria pública.

Hash SHA256

Algoritmo criptográfico que gera um identificador único de 64 caracteres para garantir integridade e imutabilidade dos documentos.
